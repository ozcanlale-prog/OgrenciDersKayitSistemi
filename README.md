# 🎓 Öğrenci Ders Kayıt Sistemi Arayüzü Projesi

## 📌 Proje Hakkında

Bu proje, İletişim ve Sunum Teknikleri dersinde öğrenmiş olduğum modern arayüz tasarımı, görsel düzen ve kullanıcı etkileşimi prensiplerini uygulamak amacıyla geliştirilmiştir. NetBeans IDE kullanılarak Java Swing teknolojisi ile bir Öğrenci Ders Kayıt Sistemi'nin kullanıcı arayüzü tasarlanmıştır.

Bu çalışma, öğrencilerin ders seçimi ve kayıt işlemlerini simüle eden bir sistemin temel arayüz yapısını oluşturmayı hedeflemektedir. Proje sayesinde masaüstü uygulama geliştirme süreci, arayüz bileşenlerinin yönetimi ve kullanıcı odaklı tasarım prensipleri uygulamalı olarak deneyimlenmiştir.

---

## 🎯 Projenin Amacı

Bu projenin geliştirilmesindeki temel amaçlar:

- Java Swing kullanarak masaüstü uygulama arayüzü geliştirme becerisi kazanmak  
- NetBeans GUI Builder (Drag & Drop) aracını etkin şekilde kullanmak  
- Kullanıcı dostu ve anlaşılır bir arayüz tasarlamak  
- Öğrenci bilgi sistemlerinin temel arayüz mantığını anlamak  
- Yazılım geliştirme sürecinde görsel tasarım prensiplerini uygulamak  
- GitHub kullanarak proje yönetimi ve versiyon kontrolü deneyimi kazanmak
